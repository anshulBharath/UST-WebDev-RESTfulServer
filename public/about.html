About the Project

        This website was created using the Leaflet API, Nominatim API, and a custom API we created for St. Paul crime data all on the Vue Framework.
        Leaflet is the API that we used to utilize different map functions. With this API we were able to create markers on the map and get specific latitudes and longitdudes of specific areas. Leaflet has a lot of functionality and customization, but a large learning curve.
        Nominatim is an API used to get locations and addresses. This was mainly used in our search bar, so that we could convert street addresses into latitudes and longitudes as per the specifications of Leaflet.
        Our custome API is a REST API that is attached to a crime database. You can provide different options in order to query different crimes. This API was used for populating our table and filtering our results to the table.
        Vue.js was the client-side framework that we used to build our front-end and handle data that was going from our client to our app. Vue made our lives a whole lot easier when creating this site and gave us an introduction into how powerful client-side frameworks are.
